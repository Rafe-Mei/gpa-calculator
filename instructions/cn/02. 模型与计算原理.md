## 1. 平均分

#### 1.1 算数平均分 (Arithmetic Mean Score)

用于衡量学生在忽略课程重要性（学分）情况下的平均得分水平。

- 公式：
 $$\large Arithmetic \; Mean = \frac{\sum_{i=1}^n \; G_i} {n}$$
 - 其中 $G_i$ 为第 i 门课程的百分制分数，n 为课程总门数。

#### 1.2 加权平均分

多数学校教务系统中核心指标，反映了学生在不同权重（学分）课程下的综合表现。

- 公式:
$$\large Weighted\; Average = \frac{\sum_{i=1}^n \; (G_i \times C_i)} {\sum_{i=1}^n \; C_i}$$
- 其中 $C_i$ 为第 i 门课程的学分

---
## 2. 总 GPA 通用算法

无论选择哪种算法模型，最终的总 GPA 都是按照学分加权计算的。

- 公式：
$$\large Total \; GPA = \frac{\sum_{i=1}^n \; (GP_i \times C_i)} {\sum_{i=1}^n \; C_i}$$
- 其中 $GP_i$ 为根据具体模型映射后的单科绩点。

## 3. GPA（4 分制）算法模型

#### 3.1 PKU Model

北大算法是目前中国高校申请海外名校时最常用的算法之一，具有极高的细分区分度。

- 分数段-GP 映射表：

| 分数段      | GP  |
| -------- | --- |
| 90 - 100 | 4.0 |
| 85 - 89  | 3.7 |
| 82 - 84  | 3.3 |
| 78 - 81  | 3.0 |
| 75 - 77  | 2.7 |
| 72 - 74  | 2.3 |
| 68 - 71  | 2.0 |
| 64 - 67  | 1.5 |
| 60 - 63  | 1.0 |
| < 60     | 0   |

#### 3.2 WES Model （Non-Official）

对 WES 评估模型的非官方近似模拟。WES（World Education Services）通常用于赴美留学认证。本程序采用其基础四档映射逻辑。

- 分数段-GP 映射表：

| 分数段      | GP  |
| -------- | --- |
| 85 - 100 | 4.0 |
| 70 - 85  | 3.0 |
| 82 - 84  | 2.0 |
| < 60     | 0   |

#### 3.3 Standard 4.0 Model

最为严格的四分制映射，常见于对成绩单有标准化硬性要求的机构。核心映射逻辑是从及格分数开始，按 10 分为一个档位均匀分布。

- 分数段-GP 映射表：

| 分数段      | GP  |
| -------- | --- |
| 90 - 100 | 4.0 |
| 80 - 89  | 3.0 |
| 70 - 79  | 2.0 |
| 60 - 69  | 1.0 |
| < 60     | 0   |

## 4. GPA（5 分制）算法模型

5 分制是中国大陆部分高校内部保研或评优时使用的标准。

#### 4.1 Standard Linear Model

该算法认为成绩与绩点呈完全线性关系，及格起算。

- 公式：
$$\large GP=max( 0 , \frac{G-50}{10} )$$

#### 4.2 Step Mapping Model

部分 985/211 高校采用的粗粒度 5 分制映射。

- 分数段-GP 映射表：

| 分数段      | GP  |
| -------- | --- |
| 90 - 100 | 5.0 |
| 80 - 89  | 4.0 |
| 70 - 79  | 3.0 |
| 60 - 69  | 2.0 |
| < 60     | 0   |
