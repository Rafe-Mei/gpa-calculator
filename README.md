# GPA Calculator

- Version: 1.0
- Author: Rafe Mei
- License: MIT


## 1. Project Overview

This is a simple and easily extensible GPA calculator that allows you to calculate average scores and GPA using various mainstream algorithms from raw percentage scores. It is suitable for undergraduate grade organization, GPA estimation for study abroad, and comparison of different calculation models.


## 2. Features

- Supports multiple GPA calculation models (PKU / WES / Linear, etc.)
- Supports GPA calculation by term or overall
- Imports raw percentage scores via CSV files
- Easy to add new GPA mapping models and calculation methods


## 3. Environment

- Python: 3.13.6 (tested)


## 4. Usage & Calculation Models

You can open the `instructions` folder to see the complete user guide and the principles of the GPA calculation models.

Default supported calculation standards:

1. Average Scores
	- Weighted Average
	- Arithmetic Mean
2. GPA (4-point scale)
	- PKU Model
	- WES Model (Non-official, Approximation)
	- Standard 4.0 Model
3. GPA (5-point scale)
	- Standard Linear Model
	- Step Mapping Model
